# Chức Năng Chính

## Bản đồ du lịch (OpenStreetMap)

- Hiển thị địa điểm du lịch nổi bật
- Route chỉ đường thông minh
- Hỗ trợ zoom/pan, clustering marker

## Tích hợp thời tiết theo thời gian thực (OpenWeather)

- Nhiệt độ hiện tại
- Độ ẩm – mưa – gió – tầm nhìn
- Dự báo + cảnh báo

## Chatbot AI (du lịch + thông tin thành phố)

- Gợi ý địa điểm
- Giải thích văn hoá – ẩm thực – sự kiện Cần Thơ
- Tra cứu triều cường và đường ngập
- Tích hợp RAG dựa trên dữ liệu JSON nội bộ
- Kết nối OpenAI API

## Admin Dashboard

- Giám sát thời tiết – mực nước – lưu lượng giao thông
- Quản lý người dùng
- Quản lý hướng dẫn viên
- Quản lý nội dung
